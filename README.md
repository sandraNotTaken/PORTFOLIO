# Portfolio Website

A simple static portfolio website containing `index.html`, `style.css`, `script.js`, and an `images/` folder.

## About

This repository is a lightweight personal portfolio built with HTML, CSS, and JavaScript. It demonstrates a responsive layout, basic interactivity, and assets organization.

## Files

- `index.html` — main page
- `style.css` — styles
- `script.js` — scripts
- `images/` — image assets

## Usage

1. Open `index.html` in your browser to view the site.
2. (Optional) Run a local static server for development:

```
python -m http.server 8000

# then open http://localhost:8000
```

## Development

- Edit `index.html`, `style.css`, and `script.js` as needed.
- Add or replace images in `images/`.
